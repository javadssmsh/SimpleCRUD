# Simple CRUD with JPA Relationships

This project is a **simple CRUD application** built to demonstrate how to work with **JPA/Hibernate entity relationships** in a real-world Spring Boot application.

The main goal of this repository is **educational**: to clearly show how **One-to-One**, **One-to-Many**, and **Many-to-Many** relationships are modeled, persisted, and exposed through RESTful APIs.
